# Archived Documentation

Documents in this directory are **historical artifacts** from the pre-v1beta1 compatibility era.
They are retained for traceability but are **not authoritative** for current development.

The kernel migrated from a compatibility bridge (ContextService) to a clean node-centric
contract (v1beta1) in PR #57. These documents describe the compatibility layer that was
removed.

For current documentation, see [docs/index.md](../index.md).
